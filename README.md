# ActiveDirectoryLab

<h2>Description</h2>
Project consists of creating a Home Lab running Active Directory Lab using VirtualBox. We will then use PowerShell to add Users, and ensure everything is functional.
<br />


<h2>Languages and Utilities Used</h2>

- <b>VirtualBox</b>
- <b>PowerShell</b> 

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
